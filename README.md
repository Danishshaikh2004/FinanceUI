# 💰 Finance Dashboard UI

A clean and responsive finance dashboard built using **React** and **Tailwind CSS**.
This project demonstrates core frontend skills like state management, data visualization, filtering, and role-based UI.

---

## 🚀 Features

* 📊 **Dashboard Overview**

  * Total Balance, Income, and Expenses cards
  * Line chart for balance trends
  * Pie chart for category-wise expenses

* 📋 **Transactions Management**

  * View all transactions in a table
  * Filter by type (Income/Expense)
  * Search by category

* 🔐 **Role-Based UI**

  * Viewer: Read-only access
  * Admin: Can add new transactions

* 📈 **Insights**

  * Highest spending category
  * Total expenses summary

* 💾 **Local Storage Support**

  * Transactions persist after page refresh

* 🎯 **UI/UX**

  * Clean modern design
  * Responsive layout
  * Fixed sidebar with smooth navigation

## ⚙️ Setup Instructions

1. Clone the repository:

```
git clone https://github.com/Danishshaikh2004/FinanceUI.git
```

2. Navigate to the project:

```
cd FinanceUI
```

3. Install dependencies:

```
npm install
```

4. Run the development server:

```
npm run dev
```
